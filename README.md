# Bikeshare Data Analysis
Created: 2023-01-08
- A project for the analysis of Bikeshare data
- Includes options to breakdown by city and day of week
- Various stats regarding duration, location, and user types

## Available City Data
- Washington, D.C.
- Chicago
- New York City
